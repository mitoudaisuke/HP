Portable NIR Fluorescence Imaging

NIRFI™

NIRFI™ is a compact near-infrared fluorescence observation device for research, training, and professional use. It enables real-time fluorescence visualization and direct PC-based image capture in a portable, low-overhead format.

Key Benefits

Designed for simple deployment

USB Connectivity

Connect to a host PC with the included USB cable for a simple imaging setup.

Direct PC Capture

Images are saved directly to the host PC for review, export, and documentation.

External Display Support

Use the host PC to display the live image on a larger external monitor when needed.

Flexible Mounting

Supports tripod-based positioning for fixed-field observation and repeatable setup.

How It Works

Set up in three steps

1
Connect

Connect NIRFI™ to a host PC using the included USB cable. Standard drivers are used for straightforward setup.

2
Position

Place the device by hand or on a tripod and align it with the area of interest.

3
Observe & Capture

Visualize near-infrared fluorescence in real time and save images directly to the host PC.
