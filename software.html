Install

Manual Installation

Install or update NIRFI manually from this page.

1
Download

Download NIRFI-Mac-arm64.dmg for macOS Apple Silicon or NIRFI-Windows-x64.zip for Windows x64.

2
Replace

On macOS, open the disk image and replace the existing app. On Windows, unzip the archive and run NIRFI from the extracted folder.

3
Launch

Launch NIRFI again and confirm the version in the application. Keep the previous download if rollback testing is required.

Updates

Manual Update Notification

Future versions of the NIRFI app may read /latest.json to notify users that a newer version is available. The app should direct users back to this software download page for manual replacement.

Settings Retention

App settings should normally remain when replacing the app, assuming the app name and application data path are unchanged.

Manual Replacement

Download the platform-specific asset, replace or extract it manually, then launch NIRFI again.

Download Asset

Use the platform-specific DMG or ZIP asset listed on this page. GitHub source code zip and tar.gz archives are generated source archives, not application distribution assets.
